# FolderOpenButton

A button that should spawn a folder open dialog or a new explorer folder.

### Unfocused

```xaml
<Metrolib:FolderOpenButton />
```
![Image of FolderOpenButton, Unfocused](Unfocused.png)

### Disabled

```xaml
<Metrolib:FolderOpenButton IsEnabled="False" />
```
