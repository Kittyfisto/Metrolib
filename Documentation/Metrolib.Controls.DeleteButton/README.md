# DeleteButton

A button which can be used to delete something for good.
                Displays a trashcan.

### Unfocused

```xaml
<Metrolib:DeleteButton />
```
![Image of DeleteButton, Unfocused](Unfocused.png)

### Disabled

```xaml
<Metrolib:DeleteButton IsEnabled="False" />
```
