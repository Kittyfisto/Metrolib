# UploadButton

A button which can be used to upload something.

Displays an upward arrow with a line beneath.

### Unfocused

```xaml
<Metrolib:UploadButton />
```
![Image of UploadButton, Unfocused](Unfocused.png)

### Disabled

```xaml
<Metrolib:UploadButton IsEnabled="False" />
```
