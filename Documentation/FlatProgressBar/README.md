# FlatProgressBar

FlatProgressBar is a ProgressBar meant to visualise how a task has progressed.

```xaml
<metrolib:FlatProgressBar Minimum="0"
                          Maximum="100"
                          Value="69" />
```
![FlatProgressBar example](Default.png)


```xaml
<metrolib:FlatProgressBar IsIndeterminate="True" />
```
